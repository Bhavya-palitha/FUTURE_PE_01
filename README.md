# LocalBoost AI – Website Copy Generator

## Overview
LocalBoost AI is an AI-powered system that generates professional website content for local businesses such as salons, cafes, and gyms.

It uses prompt engineering techniques to create structured, SEO-friendly, and conversion-focused content instantly.

---

## Problem Statement
Many local businesses struggle to create effective website content due to lack of time, writing skills, and SEO knowledge.

---

## Solution
This project uses AI to automatically generate:
- Homepage content
- About section
- Services description
- Call-to-Action
- Contact details

---

## How It Works
1. User provides business details
2. AI processes the input using structured prompts
3. Generates complete website content instantly

---

## Sample Input
Create website content for a salon in Hyderabad for students.

---

## Sample Output

GlowUp Salon – Redefine Your Style & Confidence

### About
